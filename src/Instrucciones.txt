## Proyecto Java para mejorar tu lógica de programación:

Descripción:

Desarrolla una calculadora básica en Java que funcione con números decimales, binarios, octales y hexadecimales.
La calculadora debe permitir al usuario:

1. Seleccionar el sistema numérico: Decimal, binario, octal o hexadecimal.
2. Ingresar dos números: El usuario debe poder ingresar dos números en el sistema numérico seleccionado.
3. Elegir la operación: Suma, resta, multiplicación o división.
4. Realizar la operación: La calculadora debe realizar la operación matemática seleccionada con los
   dos números ingresados y mostrar el resultado en el mismo sistema numérico.
5. Convertir el resultado: La calculadora debe ofrecer la opción de convertir el resultado a cualquiera de
   los otros tres sistemas numéricos (decimal, binario, octal o hexadecimal).

Requisitos:

- Utilizar únicamente los tipos de datos primitivos de Java.
- Implementar la conversión entre cadenas y tipos primitivos, y entre tipos primitivos.
- Mostrar mensajes claros y amigables al usuario para cada interacción.

Beneficios:

Este proyecto te ayudará a mejorar tu lógica de programación en Java de diversas maneras:

- Trabajo con tipos primitivos: Dominarás el uso de los tipos de datos primitivos de Java, incluyendo sus rangos, operaciones y conversiones.
- Manejo de sistemas numéricos: Practicarás la conversión entre sistemas numéricos decimales, binarios, octales y
  hexadecimales, lo cual es fundamental para muchas aplicaciones informáticas.
- Lógica condicional: Implementarás sentencias condicionales para manejar la selección del sistema numérico,
  la operación matemática y la conversión de resultados.
- Manejo de errores: Aprenderás a identificar y manejar errores de entrada de usuario y de conversión de tipos, mejorando
  la robustez de tu código.
- Organización y modularidad:** Estructurarás tu código de manera organizada y modular, siguiendo buenas prácticas de programación.
